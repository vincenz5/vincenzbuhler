Note: This rectangle is growing and changing color because I have SVG instructions on the x and y axis, width and height, and fill.

SVG images can do this because they're not 'bitmap' images. Bitmap is a type of image file format that can map out where pixels go, with multiple bits per pixel setting up the instructions. These drawings can be dynamic and interactive. To change the image, scripting languages like Javascript can hold the instructions. Events can be encoded into these languages like 'onmouseover' or 'onlick'. CSS can also be used.
