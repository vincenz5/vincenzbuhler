Deploying a smart contract

  • http://remix.ethereum.org
  • Keep it on Google Chrome
  • Open a new file and name it the name of the contract
  • Copy and paste in the Solidity code
  • Change the compiler version on the right to what it says after "pragma Solidity" in the code
  • Click "Start to compile" on the right
  • Make sure you're logged into Metamask
  • Enter the arguments necessary under the gas information on the right bar
  • Click the shaded text button to the argument input's left

Showing off code on Etherscan.io

  • Click on "Code"
  • Click "Verify and Publish"
  • Enter the Contract name it says in the Solidity Code
  • Choose the same compiler you used in Remix
  • Enter the same Solidity code you entered in Remix
  • If you had to enter any arguments before launching the contract, go to https://abi.hashex.org/#
  • If it was a constructor, like am address of an approved token, type leave the dropdown at "Constructor", change the next dropdown to "Address", and paste in the address you approved.
  • Copy the text in the box below and paste it into "Constructor Arguments ABI-encoded" space in Etherscan.io.
  • Verify and Publish.
